# File Handling Assignment – Python

This project demonstrates Python File Handling operations including:

- Writing data to files
- Reading files using read(), readline(), readlines()
- Appending data
- Generating summary reports
- Taking user input and saving to files
- Safe file handling using os.path.exists()

## Tasks Covered
1. Write sales data to a file
2. Read file in different ways
3. Append new sales data
4. Generate summary report
5. Create product info file using user input
6. Safe file reading with error handling
7. Mini project: Discounted price report

## Technologies Used
- Python
- File Handling
- Basic OS module
